# 看图猜成语

## 游戏简介

《看图猜成语》是一款基于图片提示的成语猜谜游戏。玩家需要通过emoji组合的图片提示，从四个选项中选择正确的成语。游戏包含连击系统、时间限制和分数系统，锻炼成语知识和联想能力。

## 游戏玩法

1. **开始游戏**：点击"开始游戏"按钮
2. **查看提示**：观察emoji组合的图片提示
3. **选择答案**：从四个选项中选择正确的成语
4. **查看结果**：答对显示释义和出处，答错显示正确答案
5. **继续游戏**：点击"下一题"继续挑战

## 游戏特色

- **图片提示系统**：使用emoji组合生成图片提示，直观有趣
- **连击系统**：连续答对获得额外加分
- **时间限制**：每题30秒，增加紧张感
- **分数系统**：基础分100分，连击加分
- **最高分记录**：使用localStorage保存最高分
- **移动端适配**：响应式设计，支持各种屏幕尺寸

## 技术实现

- **纯HTML/CSS/JS**：无框架依赖，轻量高效
- **响应式设计**：使用CSS Grid和Flexbox
- **本地存储**：使用localStorage保存最高分
- **emoji提示**：使用emoji组合生成图片提示

## 游戏数据

内置50个常用成语，每个成语包含：
- 成语本身
- emoji提示（3-5个emoji组合）
- 释义
- 出处
