# 2026-03-03 调研（第4轮）

## 热门游戏发现

### Merge 游戏趋势
- **Suika Game (合成大西瓜)** | 平台：Browser/Mobile | 核心玩法：水果合成，相同水果合并成更大的水果，物理碰撞，策略布局
- **Emoji Merge** | 平台：Browser | 核心玩法：表情符号合成，类似 Suika Game 但使用表情符号
- **Merge Rot** | 平台：Browser | 核心玩法：旋转合并，创新玩法，快速上手

### 为什么 Merge 游戏适合教育化
1. **简单操作**：单指点击/拖拽，移动端友好
2. **上瘾机制**：追求更高分数，合成更大物品
3. **策略性**：需要规划布局，培养空间思维
4. **物理反馈**：物品滚动、碰撞，增加趣味性
5. **易于教育化**：可以将水果替换为数学元素（数字、几何图形等）

## 今日选题

**《数学合成塔》** — 基于 Suika Game 的合成玩法 + 数学教育

### 原版玩法参考
- Suika Game 核心：将相同水果放入容器，合并成更大的水果
- 物理引擎：水果滚动、碰撞、堆叠
- 目标：合成最大的水果（西瓜），获得高分
- 策略：合理布局，避免容器溢出

### 复刻要点
1. **核心循环**：点击/拖拽放置数学元素（数字、几何图形），相同元素合并成更大的元素
2. **教育融合**：
   - 数字合成：相同数字合并成更大的数字（如 2+2=4）
   - 几何图形合成：相同形状合并成更大的形状（如三角形→正方形→五边形）
   - 数学运算：合并时显示运算过程（如 "2+2=4"）
3. **难度递进**：
   - 初级：1-10 的数字加法
   - 中级：1-20 的数字加减法
   - 高级：1-50 的数字四则运算
4. **视觉反馈**：
   - 合并时显示运算动画
   - 元素颜色随数值变化
   - 得分特效和连击奖励
5. **成就系统**：
   - 合成最大数字（如 1024）
   - 连续合并次数
   - 总得分里程碑
6. **本地存储**：保存最高分和进度
