# Tech-Spec: 数学斜坡跑

**Created:** 2026-03-03

## Overview

### Problem Statement

如何将经典的 Slope 3D 球体滚动游戏与数学教育相结合，创造一个既有上瘾性又有教育意义的休闲游戏？目标是让玩家在享受快节奏滚动乐趣的同时，自然地进行数学练习，提高计算能力。

### Solution

基于 Slope 的 3D 球体滚动玩法，玩家通过左右倾斜设备或点击屏幕控制球体在斜坡上滚动，避开障碍物，收集分数。每前进 100 米弹出一道数学题，答对获得加速和护盾，答错减速。难度随距离递进：加法 → 减法 → 乘法 → 除法。游戏包含视觉反馈、成就系统和本地存储最高分。

### Scope

**In Scope:**
- 核心滚动机制：左右倾斜或点击控制球体在斜坡上滚动
- 障碍物生成：随机生成障碍物，需要避开
- 数学题弹出：每前进 100 米弹出一道数学题，答对奖励，答错惩罚
- 难度递进：加法 → 减法 → 乘法 → 除法，随距离解锁
- 视觉反馈：球体颜色变化，路径颜色渐变，得分特效
- 成就系统：前进 1000 米、答对 50 题等成就
- 本地存储：保存最高分和进度
- 移动端优先，响应式设计
- 纯 HTML/CSS/JS，无框架依赖

**Out of Scope:**
- 多人游戏或在线排行榜
- 复杂的 3D 图形或物理引擎
- 音效或背景音乐
- 用户账户系统
- 复杂的设置或选项菜单

## Context for Development

游戏工厂采用纯 HTML/CSS/JS 技术栈，无框架依赖。所有游戏都是独立的 HTML 文件，包含内联 CSS 和 JavaScript。游戏通常包含以下结构：

### Technical Decisions

1. **渲染方式**：使用 Canvas 2D 上下文模拟 3D 效果，因为真正的 3D 需要 WebGL，会增加复杂度。
2. **控制方式**：支持设备倾斜（DeviceOrientation API）和点击/触摸两种控制方式。
3. **数学题生成**：使用 JavaScript 随机生成数学题，根据当前难度级别调整运算符和数字范围。
4. **状态管理**：使用全局变量存储游戏状态（分数、距离、当前难度、游戏是否结束）。
5. **本地存储**：使用 localStorage 保存最高分和成就进度。
6. **响应式设计**：使用 CSS 百分比和 vw/vh 单位，确保在不同屏幕尺寸上正常显示。
